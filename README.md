# vue-theatre
### Фишки
- SPA на Vue.js. Использует vue-router
- Использует Firebase (базу данных), коннкетится к ней через fetch по адресу https://vp-teatr.firebaseio.com/repertory.json
- Использует Vue CLI v3.0
